# Primeiros Passos com Power BI 📊

Este repositório faz parte da minha trilha de aprendizado para a formação **Analista de BI** na plataforma [DIO.me](https://www.dio.me/). Aqui, documento os conceitos fundamentais, as ferramentas e os processos iniciais para a criação de dashboards inteligentes e análise de dados.

## 📌 Sobre o Módulo
Nesta etapa, o foco é compreender o ecossistema do Power BI e como transformar dados brutos em informações acionáveis.

---

## 🏗️ O Fluxo do Power BI (Workflow)
O processo de criação de um relatório segue geralmente três etapas principais dentro das ferramentas da Microsoft:

1.  **Power BI Desktop:** Onde os dados são conectados, limpos e os relatórios são criados.
2.  **Power BI Service (Online):** Onde os relatórios são publicados e compartilhados com outros usuários.
3.  **Power BI Mobile:** Para visualização dos dashboards em dispositivos móveis.



---

## 🛠️ Pilares do Desenvolvimento no Power BI

Para construir uma solução de BI robusta, trabalhamos em três frentes:

### 1. ETL (Extração, Transformação e Carga) ou ELT (Extração, Carga e Transformação)
Utilizamos o **Power Query** para conectar em diversas fontes (Excel, SQL, Web) e realizar a limpeza dos dados (tratar nulos, alterar tipos de colunas, etc).
> *"Dados limpos são a base de uma análise confiável."*

### 2. Modelagem de Dados
Nesta fase, definimos como as tabelas se relacionam. O foco é entender conceitos como:
* **Tabelas Fato:** Registros de eventos (vendas, acessos).
* **Tabelas Dimensão:** Características dos eventos (produtos, clientes, tempo).
* **Esquema Star Schema:** O padrão ouro para performance em BI.

### 3. Visualização e DAX
Criação de medidas utilizando a linguagem **DAX (Data Analysis Expressions)** para cálculos complexos e a montagem do layout visual para facilitar a leitura do usuário final.
